# EnerQuiz

Joc de trivia sobre el consum d'energia a Catalunya.

Dades obtingudes del servei de Dades Obertes de la Generalitat.

Veure la memòria per a més informació sobre el projecte.

- Laravel (backend)

Base de dades MySQL
